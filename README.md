# MineSwarmSimulator
The bee swarm minecraft mod.

This branch is the latest version of the mod (1.21.1).

DISCLAIMER: This repository is the only official place where you can download this mod.

## Current state of the mod
The mod is still currently in pre-pre-pre-pre-pre alpha, so don't expect a full release until at least a few months / years.

When it will release (someday), it will be available for fabric (also probably on Modrinth idk) on 1.21.1, and maybe other versions if I have enough time to update the mod.

Sorry for the long wait, it is mainly because of studies, and I took the time needed to think about this mod, which gave me new ideas to implement.

Don't expect frequent updates though. Thank you.

## Dependencies
Needs both Fabric API and Trinkets API to run properly
